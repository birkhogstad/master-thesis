CI/CD står for Continuous Integration/Continuous Deployment (kontinuerlig integrasjon/kontinuerlig distribusjon), og er en praksis som brukes for å automatisere bygging, testing og utrulling av programvare.

CI/CD er sentralt i DevOps, som er en tilnærming innen programvareutvikling som fremmer samarbeid mellom utviklere og driftspersonell.

Ved å bruke CI/CD kan teamet ditt raskt og effektivt levere programvare av høy kvalitet til brukerne.
Dette kan bidra til å redusere feil, øke produktiviteten og forbedre brukeropplevelsen.

På denne nettsiden vil du lære om de grunnleggende prinsippene bak CI/CD og hvordan disse kan anvendes i arbeidslivet.
Nettsiden demonstrerer hvordan CI/CD fungerer ved hjelp av verktøy som Jenkins og Docker.
